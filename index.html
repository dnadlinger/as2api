as2api

About

This is a JavaDoc-a-like for ActionScript 2.0.

It began it because there don't seem to be any complete, free (GPL or simmilar), API documentation tools, as of November 2004. The script is implemented in Ruby.

Status

Not really usable yet.

[This page is an attempt to 'land grab' the project name.]

Examples

com.xfactorstudio.xml.xpath
Documentation for the XFactor Studio XPath API.
com.asunit.framework
Documentation for the AsUnit unit-testing framework.
